# Trunk CI Issue Reproduction Repo

## How to reproduce the problem

1. Check out this repo locally
2. Install dependencies via `pnpm install`
3. Run `trunk check --all` and see that it passes
4. Check out the [CI Action](https://github.com/chapati23/trunk-ci-issue-repro/actions/runs/12397669624/job/34608593293) and see that it fails
